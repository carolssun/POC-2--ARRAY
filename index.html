Filter

A função filter em JavaScript é um método de array que cria uma nova array com todos os elementos que passam em um teste fornecido por uma função de callback. Em outras palavras, ela permite que você filtre os itens de um array com base em uma condição e retorna um novo array contendo apenas os elementos que atendem a essa condição.

Reduce

A função reduce em JavaScript é um método que faz com que vários elementos de um array sejam reduzidos para um. Nesse caso, estamos somando as idades de todos os gatinhos.

Spread

O spread é um operador representado por três pontos (...), que tem como principal objetivo facilitar a iteração com o array ao combinar, copiar ou passar os elementos.
